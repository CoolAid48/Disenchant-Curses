# Disenchant Curses
![](https://img.shields.io/badge/Loader-Fabric/Quilt_%2b_NeoForge-purple?style=for-the-badge)
![](https://img.shields.io/badge/Supports_MC_Versions-1.21.11-purple?style=for-the-badge)
![](https://img.shields.io/badge/Environments-Server-purple?style=for-the-badge)

### Remove curses from books and gear with a grindstone as you would with any other enchant!

## This mod features:
- **Ability to remove `Curse of Binding` and `Curse of Vanishing` from any piece of gear or enchanted book**
- **No experience points are rewarded when stripping a curse with a grindstone unlike other enchants**
- **Functions completely server-side and can be used in singleplayer**
#### Any bugs or issues, please make a report on [GitHub](https://github.com/CoolAid48/Disenchant-Curses/issues)
